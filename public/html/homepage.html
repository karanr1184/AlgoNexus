Algorithms

Shortest Job First

Shortest job first is a CPU scheduling algorithm in which the process with the smallest execution time is selected for execution next. Shortest job first can only be non-preemptive. Owing to its simple nature, shortest job first is considered optimal. It also reduces the average waiting time for other processes awaiting execution.

Dining Philosopher

This problem states that there are philosophers sharing a circular table and they eat and think alternatively. A philosopher needs both their right and left chopstick to eat. A hungry philosopher can eat only if there are both chopsticks available. Otherwise a philosopher puts down their chopstick and begins thinking again.

Look / C-Look

Look and C-Look Algorithms are Disk schedulling algorithms. When any process needs to access the disk memory, there must be some procedure to reach to exact position on the disk. And these algorithms provide the exact location and other information about disk, memory and and other information like seek count.

FIFO Page Replacement

FIFO page replacement algorithm is involved in memory management when new pages in a queue are demanded, to replace the existing page with the new page. When a process needs to switch the pages in between, there must be some algorithm. And this Algorithm provides the solution for it.
